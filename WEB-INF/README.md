# Prueba técnica - JSP y Servlet

Este proyecto implementa una solución basada en Java Web utilizando JSP y Servlets.

## Tecnologías
- Java 17
- Apache Tomcat
- JSP
- JavaScript
- CSS

## Funcionalidad
- Un Servlet genera un vector de datos (frutas)
- El JSP recorre el vector y llena un <select>
- JavaScript cambia el color de la opción seleccionada
- CSS gestiona los estilos

## Ejecución
1. Desplegar en Apache Tomcat
2. Acceder a: http://localhost:8080/prueba/index.jsp
